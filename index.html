HTML이란 무엇일까?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. web browser receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and origninally included cues for the appearance of the document.
